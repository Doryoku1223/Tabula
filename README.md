## ? 核心特性 (Features)

* **? 禅意模式**：拒绝焦虑，每次只推送 15 张照片，碎片时间即可完成整理。
* **? 手势操作**：借鉴卡片式交互，左滑/右滑/上滑，单手即可轻松操作。
* **?? 隐私优先**：**仅本地处理 (Local Only)**。Tabula 不需要联网权限，您的照片绝不会上传到任何服务器。
* **? 深色模式**：完美适配 Android 深色主题，夜间整理更护眼。
* **?? 后悔药机制**：所有删除操作进入“回收站”，给您二次确认的机会。
